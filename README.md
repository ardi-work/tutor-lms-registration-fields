# Tutor LMS Custom Registration Fields

WordPress plugin that adds custom registration fields to Tutor LMS student registration form.

## Description

Tutor LMS Custom Registration Fields extends the default Tutor LMS registration form by allowing administrators to add custom fields to the student registration process.

## Features

- Add custom fields to Tutor LMS registration form
- Admin interface for managing custom fields
- Customizable field types and validation
- Works with Tutor LMS student registration

## Installation

1. Upload the plugin files to `/wp-content/plugins/tutor-custom-registration-fields` directory, or install via the WordPress Plugins screen.
2. Activate the plugin through the Plugins screen in WordPress.
3. Go to Tutor LMS → Custom Fields to add your custom registration fields.

## Frequently Asked Questions

### Does this plugin require Tutor LMS?

Yes, this plugin requires [Tutor LMS](https://www.themeum.com/product/tutorlms/) to be installed and activated.

### Can I add multiple custom fields?

Yes, you can add as many custom fields as needed from the admin interface.

### What field types are supported?

The plugin supports various field types including text, email, phone, textarea, select, and more.
